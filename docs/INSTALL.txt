johnpatrick.theme Installation
------------------------------

To install johnpatrick.theme using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``johnpatrick.theme`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        johnpatrick.theme
       
* Re-run buildout, e.g. with:

    $ ./bin/buildout
